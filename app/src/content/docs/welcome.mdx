# Welcome to CCResDoc

CCResDoc is a documentation browser for your local `~/.claude/` resources.

Wave 2 (Rust generator) will populate this site with:

- **CLAUDE.md** — project and global configuration
- **Commands** — custom slash commands
- **Skills** — reusable skill definitions
- **Agents** — agent configurations

Once Wave 2 is running, content will appear in the **Claude** section of the navigation.
